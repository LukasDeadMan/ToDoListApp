# Frontend

This frontend is part of the root `ToDoApp` project.

Main details:

- React application served in development on `http://localhost:3000`
- public landing page plus authenticated task app
- session-based integration with the Flask backend
- tests executed with Jest and Testing Library

Use the repository root [`README.md`](../README.md) as the main source of truth for setup, architecture, routes, and project status.
